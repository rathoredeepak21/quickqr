About

QuickQR

QuickQR ek simple tool hai jo aapki UPI details ko use karke instant payment QR generate karta hai. Aap amount enter karke QR banaiye, scan karwaiye, aur payment receive kijiye.

Key features

  • Fast UPI QR generation
  • Quick amount suggestions (₹100 / ₹500 / ₹1000)
  • QR auto-expiry timer (10 minutes)
  • Download QR as image
  • Basic themes & background customization

Contact

Support/feedback ke liye apni contact email yahan update kar sakte ho: support@example.com

Version: 1.0.0

Privacy Policy

Summary

QuickQR aapka data primarily aapke device par hi store karta hai (browser local storage). Hum aapki UPI ID ya name ko kisi server par upload nahi karte — kyunki is app me server hi nahi hai.

Effective date: 2026-04-27

What we store

  • Name (aapka display name)
  • UPI ID (QR generate karne ke liye)
  • Theme/mode/background preferences (optional)

Ye data aapke browser ke local storage me save hota hai, taki next time app open karne par aapko dobara setup na karna pade.

What we don’t collect

  • Payments, bank details, card details
  • OTP / PIN
  • Contacts
  • Location

Third-party services

QR generation ke liye hum `qrcodejs` library CDN se load karte hain. Browser CDN se file fetch karta hai — is process me CDN ko basic request logs mil sakte hain (jaise IP address, user-agent).

Data deletion

Aap local data delete karna chahte hain to browser settings me site data clear kar sakte hain, ya phir Settings me available Reset App button ka use kar sakte hain.

Changes

Hum is policy ko update kar sakte hain. Updates yahin page par reflect honge.
